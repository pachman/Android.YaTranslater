# Android.YaTranslator
My android translator

dependencies
-----------------------------------

* ButterKnife 8.5
* Dagger 2.9
* Gson 2.8
* RxAndroid 2
* Retrofit 2
* StorIO
* Stream
* Retrolambda

Other
-----------------------------------
* Use Android API 23
* Test device on Android 6
